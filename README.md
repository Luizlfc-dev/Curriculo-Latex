# Curriculo-Latex

Este repositório contém o currículo em LaTeX de Luiz Felipe Carvalho (arquivo principal: `curriculo.tex`). O template gera um PDF limpo e profissional, pronto para envio a vagas, seleções e apresentações.

Principais pontos
- Arquivo principal: `curriculo.tex`.
- Autoria/contato no cabeçalho do arquivo: Luiz Felipe Carvalho — Campina Grande, PB — lf.codes2002@gmail.com
- Links públicos: LinkedIn e GitHub configurados no documento.

Requisitos
- Distribuição LaTeX (TeX Live, MiKTeX) com os pacotes usados no template: inputenc, fontenc, babel, geometry, microtype, enumitem, titlesec, array, tabularx, hyperref, bookmark, xurl.

Como compilar
1. Abra o terminal na pasta do repositório.
2. Rode um dos comandos abaixo:
   - pdflatex curriculo.tex && pdflatex curriculo.tex
   - ou, se disponível: latexmk -pdf curriculo.tex
3. O PDF gerado estará no mesmo diretório (normalmente `curriculo.pdf`).

Personalização
- Edite o cabeçalho em `curriculo.tex` para alterar nome, localização, e-mail, LinkedIn e GitHub.
- Atualize seções (Resumo Profissional, Experiência, Habilidades, Educação) conforme necessário.
